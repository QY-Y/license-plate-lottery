# 摇不上概率
####仅支持个人普通指标
##使用说明
 1. 你需要登陆[摇号官网](http://www.bjhjyd.gov.cn/)，下载历年的摇号数据，然后在`PY`文件边上新建一个名为`data`的文件夹，把`zip`格式的数据放进去。
 2. 使用`python count.py`运行
 3. 按照提示输入自己的编号
##结果说明
- 第一列 期编号，例如2016年第一期就是201601
- 第二列 当前期的中签概率
- 第三列 当前期你的中签倍率
- 第四列 截至当前期，你摇不上的概率
